# Week 1 - Git and Github
### Git 이란?
파일의 변경 사항을 추적하고 여러 사용자 간에 해당 파일들의 작업들을 조율하기 위한 분산 버전 관리 시스템


Working Directory : 내가 작업하고 있는 공간
Staging Area : 내 파일의 변경사항을 반영하는 공간
Local Repository : 내 작업이 기록되는 곳

### .gitignore 사용방법
1. touch .gitignore
2. "파일 명" 작성 후 저장

### Git을 사용하여 파일 추적 상태 관리
1. git add .
2. git add "파일 이름"
3. git commit -m "커밋 메시지"
4. git push origin "브랜치 이름"
